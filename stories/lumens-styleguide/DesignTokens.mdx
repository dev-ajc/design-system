## Typography

&nbsp;

### Font Families

&nbsp;

```scss
$font-family-primary: 'Roboto', 'Helvetica Neue', Helvetica, Arial, sans-serif;
$font-family-secondary: 'Merriweather', Georgia, 'Times New Roman', serif;
$font-family-monospace: 'SFMono-Regular', Consolas, 'Liberation Mono', Menlo, Courier, monospace;
```

```scss
$font-weight-light: 300;
$font-weight-regular: 400;
$font-weight-medium: 500;
$font-weight-semibold: 600;
$font-weight-bold: 700;
```

### Line Heights

```scss
$line-height-tight: 1.2;  // Headings
$line-height-base: 1.5;   // Body text
$line-height-loose: 1.8;  // Extended reading text
```

### Letter Spacing

```scss
$letter-spacing-tight: -0.025em;  // Headings
$letter-spacing-normal: 0;        // Body text
$letter-spacing-wide: 0.025em;    // Small caps, uppercase
$letter-spacing-wider: 0.05em;    // More pronounced uppercase
```

## Using Tokens in Your Code

&nbsp;

```scss
// Instead of this:
.button {
  padding: 8px 16px;
  background-color: #3498db;
  border-radius: 4px;
  transition: background-color 300ms ease-in-out;
}

// Do this:
.button {
  padding: $spacing-sm $spacing-md;
  background-color: $color-primary;
  border-radius: $border-radius-md;
  transition: background-color $transition-base $transition-timing-function;
}
```
