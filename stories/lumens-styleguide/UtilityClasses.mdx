# Utility Classes

&nbsp;

## Spacing Utilities

Spacing utilities control margins and padding to create consistent spacing between and within elements.

### Margin

Margin utilities use the format `.m{side}-{size}` where:
- `m` stands for margin
  - `{side}` can be:
    - `t` (top)
    - `r` (right)
    - `b` (bottom)
    - `l` (left)
    - `x` (horizontal)
    - `y` (vertical)
    - omitted for all sides
  - `{size}` represents predefined spacing values:
    - `0`
    - `xs` (4px)
    - `sm` (8px)
    - `md` (16px)
    - `lg` (24px)
    - `xl` (32px)
    - `2xl` (48px)
    - `3xl` (64px)

```html
<div class="m-md">All-sides margin (16px)</div>
<div class="mt-md">Top margin (16px)</div>
<div class="mr-md">Right margin (16px)</div>
<div class="mb-md">Bottom margin (16px)</div>
<div class="ml-md">Left margin (16px)</div>
<div class="mx-md">Horizontal margins (16px)</div>
<div class="my-md">Vertical margins (16px)</div>
<div class="m-auto">Auto margin (for centering)</div>
```

### Padding

Padding utilities follow the same pattern as margin utilities, but with `p` instead of `m`:

```html
<div class="p-md">All-sides padding (16px)</div>
<div class="pt-md">Top padding (16px)</div>
<div class="pr-md">Right padding (16px)</div>
<div class="pb-md">Bottom padding (16px)</div>
<div class="pl-md">Left padding (16px)</div>
<div class="px-md">Horizontal padding (16px)</div>
<div class="py-md">Vertical padding (16px)</div>
```

## Display Utilities

Display utilities control how elements are displayed in the document flow.

```html
<div class="d-none">Hidden (display: none)</div>
<div class="d-inline">Display as inline</div>
<div class="d-inline-block">Display as inline-block</div>
<div class="d-block">Display as block</div>
<div class="d-flex">Display as flex container</div>
<div class="d-grid">Display as grid container</div>
```

### Responsive Display

Each display utility also has responsive variants:

```html
<div class="d-block d-md-none">Visible on small screens, hidden on medium and up</div>
<div class="d-none d-lg-block">Hidden by default, visible on large screens</div>
```

## Responsive Utility Patterns

Most utility classes can be applied at specific breakpoints using the pattern:
`{property}-{breakpoint}-{value}`

```html
<!-- Different text alignment at different screen sizes -->
<p class="text-center text-md-left text-lg-right">
  Centered on small screens, left-aligned on medium screens, right-aligned on large screens
</p>

<!-- Different width at different screen sizes -->
<div class="w-full w-md-1/2 w-lg-1/3">
  Full width on small screens, half width on medium screens, one-third width on large screens
</div>

<!-- Show/hide based on screen size -->
<div class="d-none d-md-block">Hidden on small screens, visible on medium and up</div>
<div class="d-block d-lg-none">Visible on small and medium screens, hidden on large screens</div>
```

## Using Multiple Utility Classes Together

Utility classes can be combined to create complex layouts without writing custom CSS:

```html
<div class="d-flex flex-column flex-md-row justify-content-between align-items-center p-md gap-lg bg-light-gray rounded">
  <div class="w-full w-md-1/2 mb-md mb-md-0">
    <h2 class="text-2xl font-weight-bold mb-sm">Left Content</h2>
    <p class="text-md text-dark-gray">Some descriptive text here.</p>
  </div>
  <div class="w-full w-md-1/2">
    <div class="border border-primary rounded p-md bg-white">
      <h3 class="text-xl text-primary mb-sm">Right Content</h3>
      <p class="text-sm">Additional content here.</p>
    </div>
  </div>
</div>
```
